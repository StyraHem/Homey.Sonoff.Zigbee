Sonoff Zigbee

Homey-appen kommer att lägga till stöd för Sonoff-enheter.

Enheter som stöds:
-Sonoff RF/TX-enheter (skicka och ta emot)
-Sonoff S60ZBTPF  (Vägguttag med effektmätning)
-Sonoff SNZB-01   (Fyrkantig knapp)
-Sonoff SNZB-01P  (Rund knapp)
-Sonoff SNZB-01M  (4-i-1 knapp)
-Sonoff SNZB-02   (Fyrkantig termometer)
-Sonoff SNZB-02D  (Termometer med display)
-Sonoff SNZB-02DR2 (Termometer med display och utomhussond)
-Sonoff SNZB-02LD (Utomhustermometer med sond)
-Sonoff SNZB-02P  (Rund termometer)
-Sonoff SNZB-02WD (Termometer för utomhusbruk)
-Sonoff SNZB-03   (Rörelsedetektor)
-Sonoff SNZB-03P  (Rörelsedetektor)
-Sonoff SNZB-04   (Dörr/fönsterkontakt)
-Sonoff SNZB-04P  (Dörr/fönsterkontakt)
-Sonoff SNZB-05P  (Vattenläckagesensor)
-Sonoff SNZB-06P  (Närvarodetektor)
-Sonoff SWV       (Vattenventil)
-Sonoff ZBCurtain (Gardin)
-Sonoff ZBMicro   (USB-switch)
-Sonoff ZBMINI-L  (Strömbrytare)
-Sonoff ZBMINI-L2 (Strömbrytare)
-Sonoff ZBMINIR2  (Strömbrytare med extern knapp)
-Sonoff ZBM5      (Väggströmbrytare, 1/2/3 kanaler)
-Sonoff ZBTRV     (Radiatorventil)
